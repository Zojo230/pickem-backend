# Shama-lama-ding-dong Football Challenge - Backend

## Setup Instructions

1. Open Command Prompt (press Windows key, type 'cmd', press Enter)
2. Navigate to this folder:
   cd Desktop\Program_by_ChatGPT_June_7_2025_Backend
3. Run the following:
   npm install
4. Then start the server:
   npm start

Your backend server will run at:
http://localhost:4000

Your frontend app will be able to send and receive picks through this server.
